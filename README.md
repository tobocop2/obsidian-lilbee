# obsidian-lilbee gh-pages asset branch

This branch is the asset store for the obsidian-lilbee marketing site
and README. The README and site at `tobocop2.github.io/obsidian-lilbee/`
hot-link images from `demos/` here via:

  https://raw.githubusercontent.com/tobocop2/obsidian-lilbee/gh-pages/demos/<name>.{gif,png}

Refresh assets via `bash demos/_publish.sh` from the main worktree.
