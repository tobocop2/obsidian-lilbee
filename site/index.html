A private research assistant for everything in your vault.

Ask anything across your notes, PDFs, ebooks, code, scans — 150+ formats in all. Every answer cites its sources; click any citation to read the original passage in place. Everything stays on your machine.

what it does

encyclopedia of your vault

Index every note, PDF, ebook, and code file in your vault. Every answer points back to the line it came from — your own private Encarta, built from documents you already have.

verify at the source

Every citation in chat or wiki opens the source document scrolled to the exact passage, with the cited lines highlighted. One click, every time. Useful anywhere a confident-sounding answer needs proof.

auto-generated wiki

Concept and entity pages, written automatically from your documents and verified against the sources they cite. Drafts queue for review. Pages plug into Obsidian's graph view alongside your notes.

scans and photos

Scanned PDFs and photographed pages are converted into searchable text — tables, columns, and layout preserved.

install
install: tobocop2/obsidian-lilbee

lilbee is in beta, so it ships through BRAT — the standard helper Obsidian users install for beta plugins. Install BRAT from Settings → Community plugins → Browse, then run BRAT: Add a beta plugin and paste the slug above.

The plugin downloads the lilbee engine on first launch — there's nothing to install separately. New release every few days; you'll get them automatically.
