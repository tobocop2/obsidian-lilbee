 talk to your vault — local, private,
 grounded in your notes

────────────────────────────────────────────

 > chat with your vault via Ollama
 > semantic search across notes, PDFs, code
 > attach anything — PDFs, images, 150+ code langs
 > OCR for scanned documents
 > inline progress with cancel support
 > save conversations back to your vault
 > runs local — nothing leaves your machine

──── built with lilbee ─────────────────────

 lilbee ··················· local knowledge base backend

──── install ────────────────────────────────

 1. Install BRAT from Community Plugins
 2. BRAT → Add Beta Plugin →
    tobocop2/obsidian-lilbee
 3. Enable "lilbee" in Community Plugins
 4. Install Ollama + lilbee CLI:

 $ pip install lilbee
 $ cd /path/to/vault
 $ lilbee init && lilbee serve
