 > ask questions, get answers from your docs
 > search across notes, PDFs, and code
 > attach anything — PDFs, images, Word, Excel
 > read scanned documents and images with OCR
 > save conversations back to your vault
 > one-click model download and server updates
 > 100% local — nothing leaves your machine

──── what you need ─────────────────────────

 1. Ollama — runs AI models locally on
    your machine. free, no account needed.

 2. Obsidian

──── install ────────────────────────────────

 1. Install + open Ollama
 2. Install BRAT from Community Plugins
 3. BRAT → Add Beta Plugin →
    tobocop2/obsidian-lilbee
 4. Enable "lilbee" in Community Plugins

 the plugin downloads the server and models
 automatically — no terminal needed.

──── built on ──────────────────────────────

 lilbee — a local-first knowledge base you
 can chat with, search, and plug into any
 tool. this plugin is built on top of it.
